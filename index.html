  • збити вершки та яйця для приготування десертів та кондитерських виробів
  • зробити повітряне суфле або мус
  • замісити рідке тісто для млинців, важке дріжджове тісто для пирогів або щільне для домашнього цільнозернового хліба

ПІД ЧАС ПРИГОТУВАННЯ

У КОМПЛЕКТАЦІЇ
ТРИ МЕТАЛЕВІ НАСАДКИ

1. Гак - насадка для замішування тіста
Режим швидкості роботи: 1-2

Призначена для важкого дріжджового тіста, для пирогів та випічки, для щільного тіста, тіста для домашнього хліба та крутого піщаного тіста.

2. Вінчик - насадка для збивання
Режим швидкості роботи: 4-6

Призначена для швидкого та ефективного збивання вершків, яєць, приготування мусів, суфле, а також домашнього морозива.

3. Лапатка (бітер) - насадка для змішування
Режим швидкості роботи: 1-3

Призначена для бісквітного тіста, змішування начинок для кондитерських виробів, солодких та пряних соусів, вимішування фаршу із зеленню та прянощами, різних салатів

ОПТИМАЛЬНИЙ РЕЖИМ
ДЛЯ БУДЬ-ЯКОГО ЗАВДАННЯ

Шість режимів швидкості: дають змогу гнучко й точно обирати потрібну потужність для будь-якого завдання, від повільного й ретельного змішування до швидкого збивання.
Імпульсний режим: для коротких сплесків потужності, які можуть бути корисними для швидкого збивання або змішування.

Міксер має захист від перегріву - систему безпеки для запобігання пошкодження двигуна внаслідок перегріву
